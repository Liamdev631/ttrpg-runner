# Mistborn Source Manifest

This pack imports the full markdown corpus currently published in the upstream repository and preserves the same top-level file separation used there.

## Imported Source Tree

- `references/README.md`
- `references/Base Game/`
- `references/Nobles - The Golden Mandate/`
- `references/Skaa - Tin & Ash/`
- `references/Terris - Wrought of Copper/`

## Import Policy

- Every imported markdown file carries a citations block at the top naming the upstream repository and the original file path.
- Imported files are copied as reference material, not as session seed libraries.
- The imported tree is preserved with the same directory split as the source repo so links, supplements, and scope remain legible.

## Era Scope

- The imported source tree is primarily Era 1 material.
- The upstream `README.md` states Era 2 expansions are intended but not yet added to the repository snapshot imported here.
- Because of that, this pack enforces Era 2 play mainly through `era-rules.md` and pack instructions rather than through a separate imported Era 2 rules directory.

## Obsidian Syntax Note

- Many imported files use Obsidian-style wiki links such as `[[Metals]]` and embeds such as `![[Complications]]`.
- Treat those as cross-references, not executable syntax.
- Prefer reading the referenced file directly when an embedded section is important to the current turn.
